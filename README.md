# 💝 Tia Mimi - Expense Management System

A custom web application developed for family expense tracking, featuring cloud integration and a mobile-friendly interface.

## 📂 Project Structure

* **[🚀 v7.1 - Cloud Edition](./v7.1-Cloud):** Latest version hosted on Google Apps Script with real-time Google Sheets integration.
* **[📦 v6.3 - Offline Edition](./v6.3-Offline):** Legacy standalone HTML version for local use.

---

## 🛠️ Roadmap & Upcoming Features
I am currently planning the following enhancements to improve the system:

### 📈 Financial Analytics
- [ ] **Total Spent Dashboard:** A real-time summary at the top of the history tab showing the total sum of expenses.
- [ ] **Monthly Breakdown:** Automatic calculation of "Total of the Month" to track budget limits.
- [ ] **Category Charts:** Visual representation (pie charts) of where the money is going.

### 📱 User Experience (UX)
- [ ] **Dark Mode:** A toggle for better night-time viewing.
- [ ] **Multi-Currency Support:** Ability to switch between ARS and USD for specific items.
- [ ] **Delete Button:** Allow users to remove a wrong entry directly from the app (with a confirmation prompt).

### ⚙️ Technical Improvements
- [ ] **Auto-Backup:** Weekly script to backup the Google Sheet into a CSV file in this repository.
- [ ] **Loading Spinners:** Better visual feedback while the app is communicating with the server.
