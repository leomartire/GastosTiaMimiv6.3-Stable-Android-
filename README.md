# 💝 Tia Mimi - Expense Management System

A simple custom web application developed for family expense tracking.
Easy to use for non Tech people. 
Financial logic and cloud integration.
This initial version is only applicable for mobiles based on Android.

## 📂 Repository Structure

### [🚀 v7.1 - Cloud Edition (Latest)](./v7.1-Cloud)
*   **Architecture:** Google Apps Script (GAS) Backend + HTML5/CSS3 Frontend.
*   **Database:** Google Sheets API.
*   **Key Features:** Real-time history, Toast notifications, and mobile-responsive UI.

### [📦 v6.3 - Offline Edition](./v6.3-Offline)
*   **Architecture:** Standalone HTML file.
*   **Usage:** Local execution via browser.
